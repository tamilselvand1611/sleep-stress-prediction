# Sleep Stress Prediction
Tool : Python, Jupyter Notebook

Dataset : Sleep Health and Lifestyle Dataset (Kaggle)

## Project Description
Created a Multiple Linear Regression model to predict a person's stress level based on lifestyle and health-related factors.

## Features Used
- Sleep Duration
- Physical Activity Level
- Heart Rate
- Daily Steps

## Target Variable
- Stress Level

## Key Results
R² Score : 0.8044
The model explains approximately 80.44% of the variation in stress levels, indicating a strong predictive relationship between lifestyle factors and stress.

## Model Insights
Sleep Duration:
- Longer sleep duration is generally associated with lower stress levels.

Physical Activity Level:
- Regular physical activity helps reduce stress.
Heart Rate:
- Higher heart rates may indicate increased stress levels.
Daily Steps:
- Active individuals tend to experience lower stress levels compared to less active individuals.

## Prediction Example
Input Values:
- Sleep Duration = 7 Hours
- Physical Activity Level = 50
- Heart Rate = 72
- Daily Steps = 8000

Predicted Stress Level:
- 5.98

Stress Category:
- Moderate Stress
